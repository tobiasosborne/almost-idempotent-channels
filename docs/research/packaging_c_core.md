# Packaging research — the C core as a distributable library

> Research leg 1 of 3 (2026-05-31). Sources cited inline. Companion docs:
> `packaging_julia.md`, `packaging_python_and_release.md`. Bead: see the
> "Professional polyglot packaging" epic. This is the **linchpin**: the CMake
> build defined here is what the Julia JLL recipe and the Python
> scikit-build-core build both consume, and it independently fixes the slow /
> hang-prone `make test`.

## The project being packaged

A tight, certified-numerics **C11 core** implementing the algorithms of
arXiv:2405.02434. Two numeric paths: a fast double/LAPACK path and an
arbitrary-precision **FLINT ≥ 3.0** (arb/acb) certified path. Native deps:
**FLINT/arb + LAPACK/BLAS**. ~80 source files (`src/*.c`, ≤200 LOC each), clean
public headers in `include/`. Currently a hand-written Makefile (auto-globs
`src/*.c`, builds test binaries + `build/libaic.so`).

## Decisions

| Decision | Chosen | Rejected | Why |
|---|---|---|---|
| Build system | **CMake ≥ 3.24** | Meson | CMake package-config is consumed natively by Julia BinaryBuilder; FLINT/SuiteSparse/GSL ecosystem; `find_package()` everywhere. (Meson is nicer DSL and is right for the *Python binding layer* — python-flint uses it — but the shared C core wants CMake.) |
| FLINT discovery | `pkg_check_modules(FLINT REQUIRED IMPORTED_TARGET flint>=3.0)` | FetchContent / submodule | FLINT is large, minutes to build, system-packaged everywhere; arb is **inside** FLINT 3.x (no separate libarb). python-flint confirms pkg-config is the hook. Assert the `>=3.0` lower bound loudly. |
| LAPACK/BLAS | `find_package(LAPACK REQUIRED)` + `find_package(BLAS REQUIRED)` | hardwire OpenBLAS | lets Julia's libblastrampoline swap BLAS at runtime |
| Symbol control | `-fvisibility=hidden` + `GenerateExportHeader` (`AIC_EXPORT`) | linker `.map` version script | simpler for a 0.x lib; add a `.map` only at 1.0 when evolving ABI within a major. FLINT/GSL use visibility attrs without symbol versioning |
| SONAME | `VERSION 0.1.0 SOVERSION 0` | — | start SOVERSION 0, bump to 1 at first API freeze (FLINT/GMP/MPFR pattern) |
| Test runner | **CTest `--parallel`** with explicit `add_test` | the `make test` serial glob loop | parallel; **eliminates the hanging-probe class** (no `tests/test_*.c` auto-exec) — directly fixes the velocity pain |
| Vendoring FLINT | **never** (for dev) | — | require system-installed; bundle only inside final artifacts (JLL tarball / Python wheel), and BinaryBuilder/auditwheel do that, not CMakeLists |

## Recommended layout

```
almost-idempotent-channels/
├── CMakeLists.txt              # root: PROJECT_VERSION, deps, install/export
├── cmake/
│   ├── AICConfig.cmake.in      # find_package(AIC CONFIG)
│   └── FindAIC.cmake           # fallback for non-config consumers
├── include/aic/                # public headers; umbrella aic/aic.h
│   ├── aic.h                   # #includes all module headers
│   └── aic_export.h            # GENERATED by GenerateExportHeader
├── src/        CMakeLists.txt + *.c
├── tests/      CMakeLists.txt + test_*.c   (add_test per binary)
├── aic.pc.in                   # pkg-config template
├── julia/AlmostIdempotentChannels.jl/
└── python/                     # meson-python OR scikit-build-core binding
```

The Julia and Python sub-trees are **not** built by the C CMake. The C build
installs `libaic.so` + headers + `aic.pc` + `AICConfig.cmake` to a prefix; Julia
`ccall`s the JLL/`Overrides.toml` library; Python's build finds it via
`PKG_CONFIG_PATH`. One CMake tree, three consumers (the Nemo.jl + FLINT_jll +
python-flint pattern).

## Root `CMakeLists.txt` skeleton

```cmake
cmake_minimum_required(VERSION 3.24)
project(AIC VERSION 0.1.0 LANGUAGES C)

set(CMAKE_C_STANDARD 11)
set(CMAKE_C_STANDARD_REQUIRED ON)
set(CMAKE_C_VISIBILITY_PRESET hidden)
set(CMAKE_VISIBILITY_INLINES_HIDDEN TRUE)

find_package(PkgConfig REQUIRED)
pkg_check_modules(FLINT REQUIRED IMPORTED_TARGET "flint>=3.0")
find_package(LAPACK REQUIRED)
find_package(BLAS REQUIRED)

add_subdirectory(src)

include(CTest)
if(BUILD_TESTING)
  add_subdirectory(tests)
endif()

include(GNUInstallDirs)
include(CMakePackageConfigHelpers)
configure_package_config_file(cmake/AICConfig.cmake.in
  ${CMAKE_CURRENT_BINARY_DIR}/AICConfig.cmake
  INSTALL_DESTINATION ${CMAKE_INSTALL_LIBDIR}/cmake/AIC)
write_basic_package_version_file(${CMAKE_CURRENT_BINARY_DIR}/AICConfigVersion.cmake
  VERSION ${PROJECT_VERSION} COMPATIBILITY SameMajorVersion)
install(FILES ${CMAKE_CURRENT_BINARY_DIR}/AICConfig.cmake
              ${CMAKE_CURRENT_BINARY_DIR}/AICConfigVersion.cmake
        DESTINATION ${CMAKE_INSTALL_LIBDIR}/cmake/AIC)
configure_file(aic.pc.in ${CMAKE_CURRENT_BINARY_DIR}/aic.pc @ONLY)
install(FILES ${CMAKE_CURRENT_BINARY_DIR}/aic.pc
        DESTINATION ${CMAKE_INSTALL_LIBDIR}/pkgconfig)
```

`src/CMakeLists.txt`:

```cmake
file(GLOB AIC_SRCS "*.c")
add_library(aic SHARED ${AIC_SRCS})
add_library(aic_static STATIC ${AIC_SRCS})
set_target_properties(aic_static PROPERTIES OUTPUT_NAME aic)

include(GenerateExportHeader)
generate_export_header(aic BASE_NAME AIC
  EXPORT_FILE_NAME ${PROJECT_BINARY_DIR}/include/aic/aic_export.h)

set_target_properties(aic PROPERTIES VERSION ${PROJECT_VERSION}
                                     SOVERSION ${PROJECT_VERSION_MAJOR})
target_compile_options(aic PRIVATE
  -Wall -Wextra -Wpedantic -Wshadow -Wstrict-prototypes -O2 -g)   # per CLAUDE.md
target_include_directories(aic PUBLIC
  $<BUILD_INTERFACE:${PROJECT_SOURCE_DIR}/include>
  $<BUILD_INTERFACE:${PROJECT_BINARY_DIR}/include>
  $<INSTALL_INTERFACE:${CMAKE_INSTALL_INCLUDEDIR}>)
target_link_libraries(aic PRIVATE PkgConfig::FLINT LAPACK::LAPACK BLAS::BLAS)

install(TARGETS aic aic_static EXPORT AICTargets
  LIBRARY DESTINATION ${CMAKE_INSTALL_LIBDIR}
  ARCHIVE DESTINATION ${CMAKE_INSTALL_LIBDIR})
install(DIRECTORY ${PROJECT_SOURCE_DIR}/include/aic
        DESTINATION ${CMAKE_INSTALL_INCLUDEDIR})
install(FILES ${PROJECT_BINARY_DIR}/include/aic/aic_export.h
        DESTINATION ${CMAKE_INSTALL_INCLUDEDIR}/aic)
install(EXPORT AICTargets NAMESPACE AIC::
        DESTINATION ${CMAKE_INSTALL_LIBDIR}/cmake/AIC)
```

`tests/CMakeLists.txt` (replaces the serial loop):

```cmake
file(GLOB TEST_SRCS "test_*.c")
foreach(src ${TEST_SRCS})
  get_filename_component(name ${src} NAME_WE)
  add_executable(${name} ${src})
  target_link_libraries(${name} PRIVATE aic)
  add_test(NAME ${name} COMMAND ${name})
endforeach()
# run:  ctest --test-dir build --parallel $(nproc) --output-on-failure
```

`aic.pc.in`:

```
prefix=@CMAKE_INSTALL_PREFIX@
libdir=${prefix}/@CMAKE_INSTALL_LIBDIR@
includedir=${prefix}/@CMAKE_INSTALL_INCLUDEDIR@
Name: AIC
Description: Almost-idempotent channels C core (FLINT/arb + LAPACK)
Version: @PROJECT_VERSION@
Requires: flint >= 3.0
Libs: -L${libdir} -laic
Cflags: -I${includedir}
```

## Export annotation

Every public function gets `AIC_EXPORT` (from the generated `aic_export.h`);
unannotated symbols are hidden:

```c
#include "aic/aic_export.h"
AIC_EXPORT void aic_factorize_build(aic_factorize *out, ...);
```

## "Do this" checklist

1. Add root `CMakeLists.txt` (3.24) per skeleton; keep the Makefile as a thin
   wrapper or delete it.
2. Move public headers under `include/aic/`; add umbrella `aic/aic.h`.
3. `GenerateExportHeader` → annotate public API with `AIC_EXPORT`
   (`-fvisibility=hidden` global).
4. `VERSION 0.1.0 SOVERSION 0` on the shared target; install `.so` + `.a`.
5. Install `aic.pc` + `AICConfig.cmake` (both — pkg-config for Meson/Python,
   cmake-config for CMake consumers).
6. FLINT via `pkg_check_modules(... flint>=3.0)`; LAPACK/BLAS via `find_package`.
   No FetchContent/submodule for FLINT.
7. Register tests with `add_test`; run `ctest --parallel $(nproc)`. This is the
   velocity fix.
8. Windows only: `target_compile_definitions(aic PUBLIC $<$<PLATFORM_ID:Windows>:FLINT_DLL>)`.

## Exemplars to imitate

- **FLINT** — installs `flint.pc` (the working downstream hook); headers
  `include/flint/*.h` with umbrella; consumed by Julia (FLINT_jll) + Python
  (python-flint). Mirror the header layout.
- **SuiteSparse (sergiud CMake port)** — clean `*Config.cmake` generation via
  `CMakePackageConfigHelpers`.
- **GSL** — `gsl.pc` + a `FindGSL` module (model for `FindAIC.cmake`).
- **python-flint** — Meson + `pkg_check_modules` for FLINT; the model for the
  *Python binding layer* (see companion doc), not the core.

## 2025 shifts worth knowing

- arb is merged into FLINT since 2.9/3.0 — `flint>=3.0` is the only module needed.
- libblastrampoline is the Julia BLAS standard — don't link OpenBLAS directly.
- `FILE_SET HEADERS` (CMake 3.23+) is the modern installable-header mechanism.
- CMake 4.x exists; pin `cmake_minimum_required(VERSION 3.24...4.0)` to opt into
  new defaults without breaking on 3.x.
